# healthy-corner-landing-page

## Live Demo

## Features
- Fully responsive
- Dark/Light mode
- Animations
- Dynamic menu items

## Tech Stack
- Vite
- React
- Tailwind CSS
- TS

## Installation
git clone https://github.com/Felina404/healthy-corner-landing-page.git 
cd healthy-corner-landing-page
npm i
npm run dev

#build
npm run build
npm run preview
